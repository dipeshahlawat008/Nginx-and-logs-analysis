## Quickstart

1. Unzip and inspect files.
2. Install dependencies (recommended in a virtualenv):
   ```bash
   pip install -r requirements.txt
   ```
3. Run CLI analysis (example):
   ```bash
   python3 main.py --analyze --log sample.log
   ```
4. Run API server:
   ```bash
   python3 api/server.py
   ```
